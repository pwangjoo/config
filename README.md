# README.md

## `Miniconda-4.7.12` (Recent)

### initial `miniconda` configuration
```
$ conda update conda
$ conda clean -a
```

### set-up `jupyter`
```
$ conda install jupyter nb_conda_kernels jupyter_client=5.3.1 (stable)
```

### command for `miniconda`
```
$ conda (env) list
$ conda (env) remove NAME

$ conda create -n NAME
$ conda activate NAME
$ conda deactivate
```
